# BrainBox Backend

This is the backend for the BrainBox application. It is a Node.js/Express application that provides a RESTful API for managing topics and notes.

## Getting Started

To get started, you will need to have Node.js and npm installed.

1.  Install the dependencies:
    ```bash
    npm install
    ```

2.  Start the development server:
    ```bash
    npm start
    ```

## Available Scripts

-   `npm start`: Starts the development server.
-   `npm test`: Runs the tests.
-   `npm run lint`: Lints the code.
